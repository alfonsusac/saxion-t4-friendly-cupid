GXPEngine.Sprite.Sprite ( string  filename,
bool  keepInCache = false,
bool  addCollider = true 
)

Initializes a new instance of the GXPEngine.Sprite class. Specify an image file to load. Please use a full filename. Initial path is the application folder. Images will be cached internally. That means once it is loaded, the same data will be used when you load the file again.

Parameters
filenameThe name of the file that should be loaded.
keepInCacheIf true, the sprite's texture will be kept in memory for the entire lifetime of the game. This takes up more memory, but removes load times.
addColliderIf true, this sprite will have a collider that will be added to the collision manager.
